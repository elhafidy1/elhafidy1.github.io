This project focused on the design and implementation of a **Smart School IoT system**.

It involved:
- Arduino-based control board
- Bluetooth communication
- Android mobile application
- Basic IoT architecture for educational environments

The project was developed as part of a **Final Applied Project (PFA)** in IoT.
